# pubmed_chemical_crawl
Crawling PubMed Chemical Data with Paper Abstracts

# Dependencies
> pip install pyqt5 metapub pyinstaller
>

# Build
> pyinstaller build.spec
>

# Run
Please run "PubMed Crawler.exe" in "dist/main"
